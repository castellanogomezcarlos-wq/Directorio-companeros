# Directorio de Compañeros

## Instalación

1. Navega a la carpeta del proyecto descargado:
```bash
   cd directorio-companeros
```

2. Instalación de las dependencias:
```bash
   pnpm install
```


## Ejecución local del proyecto

1. Iniciar el servidor de desarrollo:
```bash
pnpm dev
```

2. Abrir en el navegador con la ruta destinada, por ejemplo:
```
http://localhost:3000
```
